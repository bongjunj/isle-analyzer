# isle-analyzer

This is a `IDE` support for ISLE language.
<https://github.com/bytecodealliance/wasmtime/tree/main/cranelift/isle>

## Right now below features has supported

* go to definition
* go to references
* hover
* highlight
* rename
* inlay hints
* auto completion
* diagnose information
